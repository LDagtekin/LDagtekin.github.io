Wat ik heb geleerd
Portfolio

Wat ik heb geleerd van het maken van mijn portfolio is dat ieder zijn eigen smaak heeft. Ik heb gekeken bij anderen hun portfolio en heb gezien dat ieder zijn eigen stijl en manier heeft om dingen te laten zien.

Ook heb ik geleerd dat het enige wat telt is dat de persoon die het heeft gemaakt het mooi vindt en dat de meningen van anderen over de pagina helemaal niet uitmaken.

Hoogstwaarschijnlijk ga ik dit portfolio nooit gebruiken, maar ik weet doordit wel hoe een portfolio eruit ziet en wat het is.

Voor het maken van de portfolio heb ik veel bootstrap gebruikt en een beetje CSS. Zelf heb ik helemaal geen Javascript gebruikt.

Wat ik heb geleerd
Schooljaar

Dit jaar heb ik veel geleerd. Ik heb nieuwe mensen leren kennen en ik heb geleerd hoe het MBO werkt.

Het belangrijkste wat ik heb geleerd dit jaar is de verschillende soorten codes. Ik heb bijvoorbeeld HTML voor het eerst gebruikt dit jaar en ben best tevreden met wat ik in 1 jaar heb geleerd. Ookal is er nog veel meer om te leren, denk ik wel dat ik redelijk veel kennis heb. Javascript daarentegen heb ik bijna niks van opgepakt, omdat ik het moeilijk vind en er niks van begrijp.

Ik heb ook leren samenwerken met anderen. Vroeger kon ik niet goed samenwerken met anderen, maar dit jaar gaat dat wel goed. Vorig jaar had ik bijvoorbeeld totaal geen vertrouwen in mijn teamgenoten, maar dit jaar is dat veranderd omdat dit jaar mensen daadwerkelijk wat deden in tegen deel van mijn jaren op Mavo en Havo.

Wat ik heb geleerd
Mezelf

Tijdens het maken van deze portfolio heb ik veel geleerd over mijzelf. Ik wist bijvoorbeeld niet waarom ik deze opleiding had gekozen of wat ik later in mijn leven wou doen.

Ik weet nu dat ik coderen best leuk vindt, omdat ik steeds nieuwe dingen ontdek en ik later als werk of hobby ook wel zou willen coderen.

Ook heb ik geleerd welke soort websites ik mooi vind en welke kleuren ik goed vind passen bij elkaar.

Waarom deze kleuren

Ik heb deze kleuren gekozen omdat ik dacht dat ze goed bij elkaar pastte.

De donker blauwe kleur had ik al in gedachte toen ik hoorde dat we zelf een website moesten maken. Ik koos deze kleur, omdat het de lievelingskleur is van mijn opa en hij altijd mijn idool was. Zelf vind ik de kleur ook best mooi.

de lichtgroene kleur had ik gekozen omdat het bij de donker blauw pastte.
